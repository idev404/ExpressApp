# Instalation
To install this app:
1. Download (node.js)[https://nodejs.org/en]
2. Open CMD**
3. CD in to the project
4. Use ``npm i``

# Starting App
**To run app use: 
``node index.js``
If everything is fine then small popup will show in bash**

# Viewing website
**To view your website navigate to 
``localhost:3000``**
